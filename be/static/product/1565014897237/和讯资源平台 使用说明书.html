和讯资源教学资源管理平台 使用说明书

 

0 序言

本系统是基于教师教学过程中管理，采集教学资源核心需求所设计开发的web应用软件，同时为学生的使用做了应用实现。

 

现将本软件功能作统一说明。

 

1 快速上手

1.1 浏览器

选择合适的浏览器支持：I E8+和chrome 58+以及firefox 52+的版本。 推荐使用谷歌Chrome浏览器和苹果电脑系统上的Safari浏览器。 谷歌Chrome浏览器下载地址：http://www.google.cn/chrome/browser/desktop/

 

1.2 设备

系统支持选择电脑端或手机端登录。 电脑端可以选择Windows系统或macOS系统

 

1.3 登录

登录地址：http://www.zypt.kidhub.cn/

首次登录可能将稍等片刻。

输入账号，密码以及对应角色登录。

 

1.4 基本布局

教师登录后：

网站管理员进入网站管理

 

2.资源检索视图

进入学科首页后，主视图为本学科的热门视频和科目文章。

上方可快捷切换视图。

2.1 检索资源

视图左侧，可直接检索在栏目中属于本科目的课程。

点击课程后，可进入课程资源检索视图。

点击某个课程之后，可见该科目下所有课程。并可以根据资源（元数据）类型快速检索。

 

右侧是本学科的热门标签。通过搜索栏，可检索本学科资源上的标签。点击热门标签，或标签搜索结果，可显示该标签下，选中课程的所有资源。

2.2 资源类型

本系统资源分为9种

  • word/ppt/pdf
  • 图片/音频/视频
  • 试题/主观题/客观题

2.3 资源详情

点击资源。可查看对应资源的详情。

 

点击收藏按钮，可将自身的资源存放到收藏夹。

 

3. 个人中心视图

3.1 我的消息

个人-我的消息，可获取个人相关的消息。（审核结果。）

 

3.2 个人资料

个人-个人资料，可获取个人资料。和基本显示设置

3.3 我的收藏

教师个人收藏分为2级，允许自定义收藏夹。根据个人偏好储存采集的教学资源

3.4 我的关注

教师之间可以相互关注，学生也可以关注老师

通过关注逻辑，用户可较为方便地订阅关注用户的动态。

 

3.5 元数据（资源）-我要上传

3.5.1 直接上传

直接上传支持pdf，word，ppt，音频，视频。可直接根据后缀判断。存入系统。

上传时可给资源打标签，添加描述。

对于视频资源，系统会进行转码。因此需要一定的时间

 

3.5.2 在线创作

在线创作可选择主观题，客观题（包括单选，多选）

编辑器主要支持主流katex语法。

 

3.5.3 导入题目

作为批量导入客观题和主观题的快捷方式，导入试题和导入试卷共用一套word模板。

3.5.4 标签

上传者可以通过搜索给资源打标签，如标签不再数据库中，可上报到总后台

 

3.6 组卷

 

 

组卷可记录历史上未完成的组卷。，用户可选择继续或重组一份试卷。

用户可选择从收藏夹或全站搜索导入资源，并设置该题目分值。

 

 

在"组卷篮子"中，可拖动题目调整顺序并调整预览

 

当预览无误后，即可提交组卷。将作为一个元数据进入到审核流程

 

3.7 我的元数据

本模块反映的是个人上传元数据之后的流向。

  • 已上传：未有审核者接单的状态
  • 已发布：审核被接单且被通过
  • 未发布：进入审核流程，但尚未有结果
  • 被驳回，进入审核流程，不被通过

 

3.8 我的审核（科目审核员权限）

本模块针对科目管理员设置，拥有科目管理员的老师，可在此接受其它老师上传上来的元数据。

审核员老师可通过"待我审核"查询到自己可以看到的元数据。可翻阅详情。如合适可直接通过审核

3.9 课程管理（部分审核员权限）

3.9.1 我的课程

所有教师可查看自己创建的课程，并点击详情进入分析

 

 

3.9.2 待审核（审核员权限）

审核员可在此查看认领的课程。

3.9.3 我的审核

认领之后即可进入审核视图，或者放弃审核

 

4. 网站后台

 

拥有网站管理员权限的老师，可进入网站管理。

网站管理可收集网站的基本情况

4.1 用户管理

管理员可在此对全站用户（老师/学生）管理。（导入，查询和修改）

 

4.2 学科

可在"学科管理“中，设置学科。

对于不同学科，其重要数据（运营情况，热门课程，banner，文章，课程和元数据）皆可进行统计和管理。

从设计者的角度来说，更希望科目管理员正在个人中心完成这个工作。

4.3 公共

公共资源和标签，需要在总后台进行管理和审核

4.4 系统设置

4.4.1 环境变量

可在此对网站进行全局的设置。

 

4.4.2 运维人员（超级管理员）

本系统的运维人员定义包括

  • 网站超级管理员（1人）

  • 后台管理员，可进入网站后台的角色。主要为分担超管的工作。

  • 网站科目管理员（每个科目都有，不限制）

     

5.学生版

本系统学生版视图主要供学生使用，教师也可在上面浏览课程。亦可在搜索界面检索课程。

登录学生端。或教师在登录后界面点击"进入前台"即可进入。

 

5.1 科目

在科目详情中。可查看课程和文章列表

 

5.2 课程

学生版中，可进入课程学习，温习，管理错题

 

 

 

所有客观题都将记录进度。

 

5.3 个人中心

 

 

个人中心基本功能：

  • 查看个人信息
  • 设置收藏课程
  • 关注感兴趣的老师
  • 设置个人资料
